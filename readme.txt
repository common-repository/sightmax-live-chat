Package for integrating SightMax live chat functionality to Wordpress

== Description ==

SightMax is an intelligent powerful live chat software solution that enables you to provide real-time live chat and monitoring for sales and support. 
Increase your online sales with proactive engagement and reduce your overall customer service costs…all at an unbelievable price.  
SightMax is available as either a installable server solution where you buy it and own it or as a OnDemand SaaS solution where you simply add the Wordpress plugin to your website and you're up and running in minutes with SightMax Live Chat.

== Installation ==

1. Upload/unpack `sightmax` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
